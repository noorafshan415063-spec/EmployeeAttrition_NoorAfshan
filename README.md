# Employee Attrition Prediction using Machine Learning

## About the Project

This project was completed as part of my Data Science Internship (Week 2).

The aim of this project is to predict whether an employee is likely to leave the company based on workplace and employee-related factors using machine learning.

## Dataset

- IBM HR Analytics Employee Attrition Dataset
- Source: Kaggle
- Records: 1470

## Tools and Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Steps Performed

- Data Loading and Exploration
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building
- Model Evaluation
- Feature Importance Analysis
- Business Recommendations

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Key Findings

- Overtime was one of the strongest factors affecting employee attrition.
- Employees with frequent business travel had a higher risk of leaving.
- Certain job roles showed higher attrition than others.
- Salary alone was not enough to explain employee attrition.
